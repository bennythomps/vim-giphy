# giphy.vim

Giphy in your vim.

## Dependencies

* \*nix(-like) OS (for `mktemp`, etc.)
* [webapi-vim](https://github.com/mattn/webapi-vim)
* [ffmpeg](https://www.ffmpeg.org/) configured with `--with-libcaca`
